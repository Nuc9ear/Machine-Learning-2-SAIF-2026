# Machine Learning (II): Methods and Applications in Finance

This repository contains homework, practice questions, theory questions, notes, and project materials for the course **Machine Learning (II): Methods and Applications in Finance**.

**Instructor:** Yiqing Lin  
**Institution:** Shanghai Advanced Institute of Finance (SAIF), Shanghai Jiao Tong University  
**Main tools:** Python, Jupyter Notebook  

## Course Description

This is an introductory course on machine learning with applications in finance. The course covers essential machine learning theory, practical implementation of algorithms, and applications to financial problems.

The course requires a strong background in statistics, applied probability, optimization, calculus, and algebra. Python is used extensively during the course to illustrate the ideas discussed in class.

## Main Topics

The course covers the following topics:

1. Introduction to Machine Learning
2. Probabilistic Tools of Machine Learning
   - Distributions
   - Maximum Likelihood
   - Optimization
3. Learning Theory
   - Bias-Variance Decomposition
   - Generalization
4. Classification
   - KNN
   - Naive Bayes
   - Support Vector Machines
   - Logistic Regression
   - ROC Curve
   - AUC
5. Regression
   - Linear Regression
   - Generalized Linear Models
   - ARIMA
6. Bayesian Theory
   - Prior and Posterior
   - Kernel Trick
7. EM Algorithm and Variational Inference
8. Dimensionality Reduction and Clustering
   - PCA
   - PAA
   - k-means
   - ESG Factor Analysis
9. Neural Networks and Deep Learning
   - Backpropagation
   - CNN
   - RNN
   - Reinforcement Learning
10. Diffusion Models
